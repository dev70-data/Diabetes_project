# Diabetes_project
# Diabetes Prediction and Analysis

This project analyzes patient health data and predicts whether a person is likely to have diabetes.

## Dataset
PIMA Indians Diabetes Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## What was done
- Loaded and cleaned healthcare data
- Analyzed diabetes distribution
- Visualized glucose levels for diabetic and non-diabetic patients
- Built a Logistic Regression model to predict diabetes
- Evaluated model using accuracy score

## Output
The model predicts whether a patient has diabetes based on medical features such as glucose, BMI, and age.
